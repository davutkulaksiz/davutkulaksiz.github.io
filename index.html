ML Interface Wizard

The ML Interface Wizard is a tool designed to automatically generate a user interface (UI) for any Machine Learning model, streamlining the process of testing and interacting with models. When model parameters or metrics are updated, the tool dynamically regenerates the front-end based on provided metadata, ensuring compatibility with the latest model changes. This UI generation facilitates easy access to model predictions, validation of model performance, and enables additional testing by providing an intuitive and accessible interface for users.

LinkedHU_CENG

LinkedHU_CENG is an online platform created for students, graduates, and faculty members of the Hacettepe University Computer Engineering Department, offering features similar to LinkedIn to foster academic and professional networking within the community.

  • Profile Creation: Users can create personal profiles to share opportunities such as scholarships, jobs, and industry-related news.
  • Opportunity Application: Users can apply for posted scholarships, internships, and job opportunities directly through the platform.
  • Networking: The platform includes a chat feature, enabling users to connect and communicate with others in the community.
  • Event and News Tracking: Users can share, follow, and keep track of news, events, and online meetings relevant to the department.

    AUTO MAIL OSINT

    AUTO MAIL OSINT is an open-source intelligence (OSINT) tool developed in collaboration with a co-intern at MySecurityAnalytics. This tool enhances email verification and social media analysis by performing the following functions:

  • Email Verification: Validates email addresses by checking the format, mailbox status, and SMTP configuration.
  • Social Media Detection: Scans the web to identify if the email is registered on specific social media platforms.
  • Provider Identification: Identifies the mailbox provider associated with the email, enhancing verification accuracy.

    CADDIT - An Online Message Board

    CADDIT is an online message board where registered members can submit content across various categories, while non-registered users can browse available content. Designed for easy access and community engagement, CADDIT offers a platform for sharing and discovering information.

    User and post data are securely stored in a database managed by SQLAlchemy, with passwords hashed for security. The platform also provides an accessible API, allowing users to retrieve public data in JSON format.
